Timers screen

This part of the documentation describes the timers screen and what you can (and sometimes cannot) do with the timers screen.

As you see, the GUI interface is designed to operate (and manage) all tasks for the connected devices and minimize the need for manual changing of configuration files etc. let alone making configuration changes at compile time.

Adding a Timer to LamPI

Adding a timer to the system is a really simple process of just a few steps:

  1. On the right side of the screen, select "Timers" button from the GUI_MENU section
  2. ON the top GUI_HEADER section select the green "+" button to add an empty timer
  3. A popup appears asking for the name of the new timer
  4. Fill in a new name for the timer and press "OK" to confirm or "Cancel" to stop
  5. The new timer will appear in the GUI_HEADER section and it can be configured (see below)
